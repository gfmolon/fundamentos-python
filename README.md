# Fundamentos em Python

Repositório com exercícios e estudos básicos em linguagem Python, desenvolvidos durante a graduação em Análise e Desenvolvimento de Sistemas na UCS.

## Conteúdos

- Lógica de programação
- Variáveis e tipos de dados
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Listas
- Tuplas
- Dicionários
- Manipulação de strings
- Entrada e saída de dados
- Resolução de problemas

## Objetivo

Consolidar fundamentos importantes de programação, raciocínio lógico, organização de dados e resolução de problemas usando a linguagem Python.
